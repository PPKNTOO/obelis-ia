Consejos para Prompts Efectivos

La clave para obtener imágenes impresionantes es un buen prompt. Aquí te damos algunos consejos para guiar a la IA:

  • Sé Descriptivo: Cuantos más detalles, mejor. Piensa en colores, texturas, iluminación y ambiente.
  • Define el Estilo: ¿Arte digital, pintura al óleo, acuarela, fotografía realista, estilo anime? Especifica el tipo de arte.
  • Añade Emoción y Ambiente: Las palabras que transmiten sentimientos o atmósferas pueden influir mucho en el resultado.
  • Usa Adjetivos y Adverbios: "Un gato perezoso durmiendo" es mejor que "un gato durmiendo".
  • Prueba y Experimenta: No tengas miedo de probar diferentes prompts y tonos hasta encontrar lo que buscas.

Preguntas Frecuentes

¿Qué tipo de imágenes puedo generar?

Puedes generar una amplia variedad de imágenes, desde paisajes realistas hasta arte fantástico, retratos, objetos y escenas abstractas. La clave está en la descripción (prompt) que proporciones a la IA.

¿Son gratuitas las imágenes generadas?

Sí, todas las imágenes generadas a través de nuestra plataforma son gratuitas para uso personal. Te proporcionamos un límite de generaciones gratuitas que puedes recargar viendo anuncios.

¿Por qué mi imagen no se generó o salió extraña?

La generación de IA puede ser compleja. Si una imagen no se genera o el resultado no es el esperado, considera los siguientes puntos:
  • El prompt puede ser demasiado ambiguo o muy complejo.
  • Intenta simplificar o detallar más tu descripción.
  • Prueba la función "Mejorar Prompt" para una descripción más elaborada.
  • Puede haber una alta demanda en la API o un problema temporal. Intenta de nuevo más tarde.
